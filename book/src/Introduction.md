# Introduction

This book covers a collection of software for CCC related work, called **ccc-tools** ([GitHub](https://github.com/Noonicknames/ccc-tools)).

**ccc-tools** currently consists of,

- [extract-cs](./extract-cs/overview.md), extract cross sections from `totalcs_*` files.
- [calc-rates](./calc-rates/overview.md), calculate effective collision rates against temperature from incident energy cross sections.
